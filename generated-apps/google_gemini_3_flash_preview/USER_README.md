# Zeeter User README

## Navigation Guide

### Authentication
To use Zeeter, you need to create an account or log in.
- **Signup**: Click "Sign up" on the navbar and enter your details (email, password, username, display name).
- **Login**: Use your registered email and password.

### Key Actions
- **Feed**: The home page displays posts from the community. You can post your own 280-character updates here.
- **Interactions**: Click the Heart icon to like a post or the Message icon to view and add comments.
- **Following**: Click on a user's name or avatar to view their profile and click "Follow" to see their updates.
- **Notifications**: Check the Bell icon for alerts about new likes, comments, or followers.
- **Search**: Use the search bar on the home page to find posts or users.
