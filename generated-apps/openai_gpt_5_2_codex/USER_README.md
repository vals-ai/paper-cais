### Key Pages & Actions
- **Home (/)**
  - Logged-in members see the personalized feed with a composer at the top.
  - Use the search bar to filter by keyword or hashtag and sort by newest/trending.
  - Like or comment on posts to create notifications.

- **Explore (/explore)**
  - Public feed for visitors and members.

- **Profile (/profile/:id)**
  - View a member’s profile, recent posts, and follower stats.
  - Follow/unfollow when signed in.

- **Notifications (/notifications)**
  - See alerts for likes, comments, and new followers.
  - Mark all as read when needed.

### Onboarding
- After signing up, complete the profile form to add a display name, bio, and avatar.
