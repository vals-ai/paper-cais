# Zeeter Social Network — User README

## Navigation guide to test the app
Open the app in a desktop browser at the provided **APP_PUBLIC_URL**.

### 1) Sign in
Use any of the seeded demo accounts (password: **Password123!**):
- **alice@zeeter.dev**
- **bob@zeeter.dev**
- **carol@zeeter.dev**

You can also create a new account via **Create account** on the login page.

### 2) Onboarding / profile setup
After creating an account, you’ll be redirected to **Onboarding** to set:
- Display name
- Bio
- Optional profile image

### 3) Home feed
Go to **Home** to:
- Write a post (≤ 280 characters)
- Sort feed: **Newest** / **Trending**
- Search by keyword or hashtag (e.g. `#welcome`)
- Like posts and open comment threads
- Edit/delete your own posts using the **…** menu

### 4) Discover
Go to **Discover** to:
- Search for other members
- Follow/unfollow them
- Click a member to view their profile and recent activity

### 5) Notifications
Go to **Notifications** to:
- See alerts for likes, comments, and new followers
- Click a notification to open the related post/profile
- Mark all notifications as read
