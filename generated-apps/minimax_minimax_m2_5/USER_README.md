# Zeeter - User Guide

## Getting Started

### Accessing the Application

The application is available at `http://localhost:23929` (or the configured port).

### Creating an Account

1. Click "Sign up" on the login page
2. Enter your email and password (minimum 6 characters)
3. Click "Create Account"
4. You'll be redirected to profile setup

### Setting Up Your Profile

After signing up, you'll be asked to complete your profile:
- **Display Name**: Required (shown to other users)
- **Bio**: Optional (up to 160 characters)
- **Avatar URL**: Optional (link to an image)

Click "Complete Setup" to finish.

## Features

### Home Feed

The home page shows posts from the community. You can:
- **Create posts**: Type in the text box at the top (max 280 characters)
- **Sort posts**: Choose "Newest" or "Trending"
- **Like posts**: Click the heart icon
- **Comment**: Click the comment icon to view/add comments

### Profile

Visit your profile to:
- See your posts
- Edit your profile (display name, bio, avatar)
- See your followers and following count

### Search

Use the search page to:
- Find users by name or bio
- Find posts by content

### Notifications

The notifications page shows:
- When someone likes your post
- When someone comments on your post
- When someone follows you

Click on a notification to view the related profile or post.

## Testing Credentials

To test the application, you can create your own account using the signup flow:
1. Go to the signup page
2. Enter any email and password
3. Complete the profile setup

No email verification is required for the MVP.
